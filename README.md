# Rubicl
## Binary clock for argos

Shows a simple binary clock in the Gnome panel. Uses the wonderful [Argos](https://github.com/p-e-w/argos/) extension for Gnome.

![rubicl](rubicl.png)

## Installation
download [rubicl.1s.argos](https://github.com/kiamazi/rubicl/raw/master/rubicl.1s.argos) and copy to ```~/.config/argos```
